# Calculator
CLI Calculator with NodeJS

## How to use
### Start the app
> - Clone the project & move to the folder
> - Run next script in console: ```npm start```

### Console interface (CLI)
> Operations: includes positive as well as negative numbers
>>  - Sum: ```+```
>>  - Subtraction: ```-```
>>  - Multiplication: ```*```
>>  - Division: ```/```
>>  - Exponential: ```^```
>>  - Logarithm to base 10: ```log()```
>>  - Square root: ```sqrt()```
